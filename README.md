# UGS Node Classification on Cora, Citeseer, PubMed Datasets

## Requirements
The following packages are needed to run the scripts:

- `python==3.6`
- `torch >= 1.4.0`
- `dgl-cu101==0.4.2`
- `scikit-learn`
- `networkx`
- `scipy`
- `matplotlib`
- `pickle`
- `argparse`
- `numpy`
- `warnings`

Install these packages using pip:

```bash
pip install -r requirements.txt
